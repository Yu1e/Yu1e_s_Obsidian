# Yu1e_s_Obsidian

Весь код написан разными версиями ИИ Claude.

## Глоссарий

[Глоссарий](https://github.com/Yu1e/Yu1e_s_Obsidian/blob/Dataview/Глоссарий%20(выгрузка%20всех%20подсветок%20в%20хранилище%20в%20список)) - код dataviewjs, который собирает алфавитный список ==подсвеченных== ==фрагментов== из всех заметок хранилища. Блок кода можно вставить прямо в заметку.
Исключает из списка быстрые клавиши и сочетания подсветки с другими видами эмфаз - например с курсивом.
Во время работы можно поставить на паузу.
Есть быстрый поиск по результатам.
Сркипт для работы внутри заметки, вставляется в отдельную, специально созданную заметку один раз.

По желанию - выгружает в отдельную заметку-сохранёнку (всегда одну и ту же, т.е. содержимое будет переписываться).
Разбивает на заголовки по алфавиту.
Можно настроить выгрузку столбцами в количестве от 1 до 10. Синтаксис столбцов - плагин [Horizontal Blocks](https://github.com/iCodeAlchemy/horizontal-blocks)❤️. Список будет скомпонован на экране слева направо, а затем сверху вниз, как на бумажной странице. Не совсем привычно, но удобно.

Каталог выгрузки, названия, заголовки и подсказки можно перенастроить под себя в коде.

## Шаблон для работы с колонками текста

[multicolumn](https://github.com/Yu1e/Yu1e_s_Obsidian/blob/Templates/multicolumn.md) - шаблон Templater, который работает в связке с плагином [Horizontal Blocks](https://github.com/iCodeAlchemy/horizontal-blocks) для отображения текста в несколько колонок, наиболее простым по синтаксису из всех аналогов. Этот шаблон сам распределяет и перераспределяет текст по колонкам. Может быть применён:
- к выделенному тексту,
- к тексту, вставленному из буфера,
- к тексту, уже отформатированному плагином:
  - как выделенному,
  - так и вырезанному и вставленному в диалоговое окно во время работы шаблона.

Можно выбирать от 2 до 5 колонок, при редактировании - изменять их количество, а также добавлять и убирать текст. Колонки могут сочетаться в заметке с обычным текстом. Также в заметке может быть несколько блоков с колонками.
